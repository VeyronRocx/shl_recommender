SHL Assessment Recommender

\

Find the perfect SHL assessments for your hiring needs

\
\
\ \
\
\

About This Project

\

\ The SHL Assessment Recommender is an intelligent system that helps hiring managers find the most suitable SHL assessments based on job descriptions, requirements, or natural language queries. It uses natural language processing and machine learning to match job requirements with the most appropriate assessments from SHL's catalog.\

\
\ \

Features

\
\
\

Smart Recommendations

\

Uses TF-IDF and cosine similarity to match job requirements with assessments in the SHL catalog.

\
\
\

Multiple Input Methods

\

Enter a natural language query, paste a job description, or provide a URL to a job posting.

\
\
\

Intelligent Analysis

\

Automatically detects skills, test types, and duration constraints from your input.

\
\
\

REST API

\

Access recommendations programmatically through a well-documented FastAPI endpoint.

\
\
\ \

How It Works

\

\ The application analyzes your input using natural language processing techniques to extract relevant information such as:\

\
    \
  • Technical skills required (Java, Python, SQL, etc.)
    • \
  • Types of assessments needed (cognitive, personality, technical)
    • \
  • Maximum duration constraints
    • \
\

\ It then uses TF-IDF vectorization and cosine similarity to match your requirements with the most suitable assessments from the SHL catalog, applying filters based on your constraints.\

Example Queries

\
\

Query: "I am hiring for Java developers who can collaborate effectively with my business teams. Looking for an assessment(s) that can be completed in 40 minutes."

\

The system will: Detect Java as a required skill, identify a 40-minute time constraint, and recommend suitable technical assessments that meet these criteria.

\
\ \
\

Query: "Hiring for an analyst position. Need cognitive and personality tests under 30 minutes."

\

The system will: Identify the need for cognitive and personality assessments with a 30-minute duration limit and provide appropriate recommendations.
